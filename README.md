# Personal Expense Tracker

A Python-based application that helps users record, categorize, and monitor their daily expenses through a simple menu-driven interface.

## Features

* Add new expenses
* Categorize expenses
* View all expenses
* Calculate total expenses
* Easy-to-use menu-driven system

## Technologies Used

* Python

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run:

```bash
python expense_tracker.py
```
